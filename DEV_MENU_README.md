# 开发菜单使用说明

## 功能概述

开发菜单是一个隐藏的工具，用于在开发过程中快速访问和修改正式实验部分的各个图表。

## 如何访问开发菜单

### 方法1：点击工具按钮

- 在页面右上角有一个🔧图标按钮
- 点击该按钮即可打开开发菜单

### 方法2：键盘快捷键

- 按 `Ctrl + Shift + D` 组合键
- 可以快速打开/关闭开发菜单

## 菜单功能

### 1. 图表分类

开发菜单按图表类型分类显示：

- **折线图 (Line Graph)** - 包含所有折线图变体
- **散点图 (Scatterplot)** - 包含所有散点图变体
- **热力图 (Heatmap)** - 包含所有热力图变体
- **网络图 (Network Diagram)** - 包含所有网络图变体
- **桑基图 (Sankey Diagram)** - 包含所有桑基图变体
- **树状图 (Treemap)** - 包含所有树状图变体
- **旭日图 (Sunburst Diagram)** - 包含所有旭日图变体
- **圆形打包图 (Circle Packing)** - 包含所有圆形打包图变体
- **目的地地图 (Destination Map)** - 包含所有地图变体

### 2. 搜索功能

- 在搜索框中输入关键词可以快速查找图表
- 支持按图表名称、类型等搜索

### 3. 过滤功能

- **Highlighting** - 显示所有高亮变体
- **Staging** - 显示所有分阶段变体
- **Tracing** - 显示所有追踪变体
- **基础图表** - 显示所有基础图表

### 4. 开发模式

- 可以开启/关闭开发模式
- 开发模式状态会保存在本地存储中
- 开启开发模式后，工具按钮会变为深色

## 图表访问方式

### Vue路由图表

- 折线图相关页面通过Vue路由访问
- 点击后会在当前页面跳转

### HTML文件图表

- 其他图表类型通过HTML文件访问
- 点击后会在新标签页中打开

## 使用建议

1. **开发时**：保持开发模式开启，方便快速访问各种图表
2. **测试时**：使用搜索和过滤功能快速找到特定类型的图表
3. **调试时**：利用新标签页打开HTML文件，方便对比和调试

## 注意事项

- 开发菜单仅在开发环境中使用
- HTML文件路径基于项目根目录
- 确保所有图表文件路径正确
- 快捷键可能与其他应用冲突，请注意

## 技术实现

- 基于Vue 3 Composition API
- 使用Vue Router进行页面导航
- 支持本地存储状态持久化
- 响应式设计，支持移动端
